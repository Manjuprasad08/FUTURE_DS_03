# Bank Marketing Campaign Analysis

## 📌 Problem Statement
The goal of this project is to analyze a bank marketing dataset to predict whether a customer will subscribe to a term deposit and identify key factors influencing customer decisions.

## 📊 Key Insights
- Customers contacted via cellular have higher conversion rates
- Duration of call strongly impacts success
- Previous campaign success increases probability of conversion

## 🤖 Model / Analysis
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning Model

## 🛠 Tools Used

- Jupyter Notebook
  

## 📈 Results

- Identified key drivers of campaign success
